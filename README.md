# Буревестник

Основная версия проекта теперь запускается из файла `burevestnik.py`.

Это оконное приложение на Python для обнаружения и сопровождения воздушных объектов на видеопотоке с камеры или из видеофайла. В интерфейсе есть таблица целей, рамки на кадре, координаты, зона интереса и отладочное меню для выбора модели и шага анализа видео.

## Установка

Нужны Python, Git и Git LFS, потому что модели лежат в репозитории через LFS.

```powershell
git clone https://github.com/rockettoom666-ops/Burevestnik.git
cd Burevestnik
git lfs install
git lfs pull
```

Создай окружение и установи библиотеки:

```powershell
py -m venv .venv
.\.venv\Scripts\Activate.ps1
python -m pip install --upgrade pip
python -m pip install -r requirements.txt
```

Если PowerShell не дает активировать окружение:

```powershell
Set-ExecutionPolicy -Scope Process -ExecutionPolicy Bypass
.\.venv\Scripts\Activate.ps1
```

## Запуск

```powershell
python .\burevestnik.py
```

После запуска можно выбрать камеру или загрузить видеофайл для теста.

## Отладка

В левой панели есть блок `Отладка`.

- выбор модели из папки `модели`;
- настройка шага анализа видео: например, `5` означает анализ каждого пятого кадра;
- показ фактического шага между анализами;
- показ времени последнего YOLO-анализа в миллисекундах.

Для камеры анализ идет каждый кадр. Для видео используется выбранный шаг, чтобы поток не превращался в слайд-шоу.

## Структура

- `burevestnik.py` - основное приложение.
- `модели/` - YOLO-модели `.pt`.
- `ресурсы/` - дополнительные ресурсы проекта.
- `презентация/` - материалы для защиты.
- `requirements.txt` - библиотеки Python.

## Ветки

Наша прежняя модульная версия сохранена в отдельной ветке:

```powershell
git switch codex/modular-prototype
```
