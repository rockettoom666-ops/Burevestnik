# Буревестник

Оконный прототип на Python для детекции воздушных объектов на видео с камеры или из файла.
Программа показывает видеопоток, рисует рамки объектов, выдает ID, координаты центра, таблицу объектов и тревожное оповещение при появлении новой цели.

## Что уже есть

- выбор веб-камеры из списка найденных устройств;
- тест на локальном видеофайле;
- детекция через обученные YOLO-модели из папки `модели`;
- переключение модели прямо в интерфейсе;
- рамки, ID объекта и координаты центра `(x, y)`;
- таблица объектов под видеопотоком;
- звуковая тревога из файла `ресурсы/тревога.wav`;
- память трекера: если объект пропал на несколько кадров и появился рядом, он остается тем же ID.

## Установка

Нужны Python и Git. Для загрузки моделей нужен Git LFS.

```powershell
git clone https://github.com/rockettoom666-ops/Burevestnik.git
cd Burevestnik
git lfs install
git lfs pull
```

Создай виртуальное окружение и поставь библиотеки:

```powershell
py -m venv .venv
.\.venv\Scripts\Activate.ps1
python -m pip install --upgrade pip
python -m pip install -r requirements.txt
```

Если PowerShell не дает включить окружение, один раз выполни:

```powershell
Set-ExecutionPolicy -Scope Process -ExecutionPolicy Bypass
.\.venv\Scripts\Activate.ps1
```

## Запуск

Основная точка запуска:

```powershell
python .\код\прототип.py
```

После запуска:

1. Нажми `Работать с камерой`, чтобы выбрать веб-камеру.
2. Или нажми `Тест на видео`, чтобы открыть видеофайл.
3. Включи отслеживание в интерфейсе.
4. При появлении нового объекта программа покажет предупреждение и проиграет звук тревоги.

## Обновление проекта

Когда напарники отправили новые изменения на GitHub:

```powershell

```

## Где что лежит

- `код/прототип.py` - запуск приложения.
- `код/буревестник/интерфейс.py` - окно, кнопки, таблица и тревога.
- `код/буревестник/детекция.py` - загрузка моделей и YOLO-детекция.
- `код/буревестник/трекер.py` - ID объектов и память трекера.
- `код/буревестник/отрисовка.py` - рамки, подписи и координаты на кадре.
- `код/буревестник/настройки.py` - пороги, классы и настройки трекера.
- `модели/` - обученные `.pt` модели.
- `ресурсы/тревога.wav` - звук тревоги.

## Важно

Не запускай старые черновые файлы из корня проекта. Основная версия сейчас запускается только через:

```powershell
python .\код\прототип.py
```
